# Student Performance Analysis

## Project Overview
This project analyzes student exam performance data to identify factors affecting academic achievement, detect at-risk students, and provide recommendations for improving educational outcomes.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Dataset
Students Performance in Exams Dataset (Kaggle)

## Analysis Performed
1. Data Exploration and Cleaning
2. Parental Education Impact Analysis
3. Test Preparation Analysis
4. Correlation Analysis
5. Gender-Based Performance Analysis
6. Total Score Distribution Analysis
7. At-Risk Student Segmentation

## Key Findings
- Higher parental education is associated with better student performance.
- Students who completed test preparation scored higher in all subjects.
- Reading and writing scores have a very strong correlation.
- Male students perform better in math, while female students perform better in reading and writing.
- 188 students were identified as at-risk.

## Visualizations
- Box Plot (Parental Education vs Scores)
- Bar Chart (Test Preparation Comparison)
- Correlation Heatmap
- Grouped Bar Chart (Gender vs Subject)
- Histogram (Total Score Distribution)
- Scatter Plot (Reading vs Math)

## Recommendations
1. Expand test preparation programs.
2. Provide targeted support for at-risk students.
3. Increase parental engagement initiatives.
